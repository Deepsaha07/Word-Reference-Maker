WordRef – Support

Version: 1.0.0

WordRef is a Word add-in that lets you paste BibTeX (e.g., from Google Scholar), store a small reference library, insert citations, and automatically format your bibliography.

Help & Issues

If you find a bug, have a feature request, or need help:

  • Visit the project on GitHub:
    github.com/deepsaha07/Word-Reference-Maker
  • Open a new issue with a short description and (if possible) screenshots.

Basic Troubleshooting

  • Try closing and re-opening the WordRef task pane.
  • Click Reload inside the add-in.
  • Restart Word if things get stuck.
